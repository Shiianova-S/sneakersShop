## SneakersMania
### MVP-версия магазина кроссовок.

### Стек проекта:
<p>Frontend: JavaScript, React, Redux, CSS, HTML</p>
<p>Backend: Node.js, Express, Sequelize, PostgresSQL, JSON Web Token (JWT)</p>
